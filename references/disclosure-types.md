# Disclosure Types

Use the checklist that matches the user request.

## Annual Report / 10-K / 20-F

Check:

- business description and segment changes
- MD&A / management discussion
- risk factor changes
- revenue, margin, cash flow, capex, inventory, receivables, debt
- customer/supplier concentration
- audit opinion and related-party issues
- capital returns, dilution, SBC, convertibles

## Quarterly Report / 10-Q / 6-K / Results Announcement

Check:

- YoY/QoQ changes
- guidance and implied run rate
- segment growth and margin
- inventory, receivables, payables, cash conversion
- one-offs and accounting changes
- management commentary versus numbers

## Earnings Call

Check:

- prepared remarks: intended narrative
- Q&A: investor concerns
- directness of answers
- repeated/new/missing language
- guidance tone
- consistency with revenue, orders, margins, inventory, cash flow

## IPO Prospectus / Listing Document

Check:

- business model and revenue quality
- customer and supplier concentration
- related parties
- gross margin and cash-flow quality
- use of proceeds
- lock-up and dilution
- peer valuation and market timing
- regulatory, competitive, and execution risks

## Analyst Research

If full report is available:

- thesis, assumptions, valuation method, forecasts, target price, risks
- compare against filings/calls
- identify what would prove it right or wrong

If full report is unavailable:

- say so explicitly
- use public summaries/rating changes/target-price changes only
- do not invent unavailable details
